# Student Performance Analysis

## Project Overview
This project analyzes student academic performance using Python and Power BI.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI

## Dataset
Student Performance Dataset from Kaggle.

## Analysis Performed
- Gender vs Average Score
- Subject-wise Performance Analysis
- Correlation Heatmap
- Parent Education Analysis
- Test Preparation Analysis
- Performance Distribution

## Key Insights
- Female students performed better in reading and writing.
- Students completing test preparation scored higher.
- Reading and writing scores showed strong correlation.
- Students with higher parental education performed better academically.

## Dashboard
Interactive Power BI dashboard created for visualization and insights generation.
